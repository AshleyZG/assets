## Cursor Cloud specific instructions

This repository (`AshleyZG/assets`) is a **static assets store** containing only PDF files (academic publications and a CV). It is used to host files that are referenced by URL from an external personal/academic website.

**Key facts:**
- No source code, no programming languages, no frameworks.
- No dependencies to install (`package.json`, `requirements.txt`, etc. do not exist).
- No build, lint, or test commands.
- No services to run.
- Typical workflow: add/update PDF files and commit/push.
